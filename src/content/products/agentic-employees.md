[Body placeholder — the model behind this card is straightforward to describe and harder to ship: you hire a named AI agent for a functional role the way you'd hire a junior person, with a defined remit, defined tool access, defined working hours, and a weekly digest that says what the agent did and why. Inbox triage, SEO monitoring, lead qualification, first-pass customer support, content research — each is a discrete role with a clear handoff back to a human for anything outside the remit.]

[The agent's loop is built around the role boundary, not the chat surface. Each "employee" has a set of allowed actions (write to this Linear project, post in this Slack channel, never send mail outside the team) and a kill-switch that any operator can pull at any time. Every action emits a human-readable rationale paragraph that lands in the agent's working log. The weekly review interface is the artifact the founder or operator opens once and reads end-to-end — what the agent did, what was paged to a human, what the agent skipped because it was outside its remit. The reliability claim is operational: agents that act inside their remit and surface everything else, not agents that improvise.]

[Audience: small companies and consultancies operating with a deliberately small headcount where adding a person is a real decision, but a $79/month role-shaped agent with the right boundaries might be obviously worth it.
