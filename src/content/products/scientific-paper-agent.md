[Body placeholder — research workflows pile up faster than reading time. A new compound, a new mechanism, a new methodology lands in a paper, and the question is the same one every time: what does this say, what does it cite, and where does it disagree with the last twelve papers on the same topic? The scientific paper agent reads the paper, pulls the claims, follows every citation back to its source, and lays the cross-references out so a researcher can see at a glance which arguments are well-supported and which are floating.]

[The agent's loop has four steps: find the paper (DOI, arXiv, PubMed), extract the structured claims, cross-reference each citation against the bibliography graph the agent maintains for the user's research domain, and produce a one-page summary with every claim linked back to the paragraph it was lifted from. The reliability claim is concrete: no citation is reported unless the agent has actually fetched and indexed the cited source; no claim is paraphrased without the original sentence available behind a hover; if the agent cannot find a citation it says so out loud. There is no fabricated DOI path. There is no "summarized as" elision.]
