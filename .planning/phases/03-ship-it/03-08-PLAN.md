|----------|-------------|
| Static assets in `/public/` → CDN edge | `og.png`, `apple-touch-icon.png`, `robots.txt`, `sitemap-index.xml` are public artifacts served same-origin from Vercel's edge. Boundary is one-way (read-only by visitors); no input crosses back. |

| Crawler/bot → robots.txt + sitemap | Crawlers read directives but do not write. Boundary protected by HTTP semantics and CDN. |
